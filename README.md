# homebrew-tap

A [Homebrew](https://brew.sh) tap for `kpod13`'s tools.

## Usage

```bash
brew tap kpod13/tap
brew install <formula>
```

## Available formulae

### claude-openai-proxy

OpenAI-compatible HTTP proxy backed by the [Claude CLI](https://claude.ai/code).

```bash
brew install claude-openai-proxy
```

Upgrade with `brew upgrade claude-openai-proxy`.

Source: <https://github.com/kpod13/claude-openai-proxy>

## How this tap is maintained

Formulae in `Formula/` are generated automatically by the source projects'
release pipelines on each tagged release — do not edit them by hand.
